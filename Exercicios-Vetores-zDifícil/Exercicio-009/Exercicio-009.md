# Exercício 009 - Ordenação e identificação de maior e menor valor

## Descrição
O programa lê **5 números inteiros**, realiza a **ordenação crescente dos valores**, exibe os números do menor para o maior e identifica qual é o maior e o menor valor do vetor.

Objetivo: treinar ordenação manual de vetor, uso de laços aninhados e manipulação de posições.

## Conceitos abordados
- Declaração de vetor (`vetor[1..5] de inteiro`)
- Laços de repetição aninhados (`para` dentro de `para`)
- Algoritmo simples de ordenação (comparação e troca)
- Uso de variável auxiliar para troca de valores
- Comparação de valores (`>`, `<`)
- Identificação de maior e menor elemento

## Como usar
1. Abra o arquivo `Ex009.alg` no VisualG.
2. Execute o programa.
3. Digite os 5 números solicitados.
4. Observe os números ordenados do menor para o maior e os valores máximo e mínimo exibidos na tela.
