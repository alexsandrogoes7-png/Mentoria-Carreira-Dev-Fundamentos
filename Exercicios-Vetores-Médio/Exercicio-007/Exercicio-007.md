# Exercício 007 - Jogo de memória com vetor

## Descrição
O programa simula um pequeno **teste de memória**.  
O usuário digita 7 números, e em seguida tenta lembrar um deles. O sistema verifica se o número informado foi digitado anteriormente.

O processo pode ser repetido até que o usuário escolha encerrar o programa.

Objetivo: treinar vetores, laços de repetição, busca de elementos e estruturas de repetição com controle de saída.

## Conceitos abordados
- Declaração de vetor (`vetor[1..7] de inteiro`)
- Laço de repetição `para`
- Estrutura de repetição `repita ... ate`
- Comparação de valores
- Busca de elemento dentro de vetor
- Controle de fluxo com caractere (`quer`)

## Como usar
1. Abra o arquivo `Ex007.alg` no VisualG.
2. Execute o programa.
3. Digite os 7 números solicitados.
4. Informe um número para verificar se ele foi digitado anteriormente.
5. Escolha se deseja continuar ou encerrar o teste.
