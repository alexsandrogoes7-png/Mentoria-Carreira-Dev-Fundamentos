# Exercício 001 - Ler, exibir e somar vetor

## Descrição
O programa lê 5 números inteiros do usuário, exibe cada número digitado e calcula a soma total.
Objetivo: treinar leitura de vetores, laços de repetição e operações matemáticas básicas em VisualG/Portugol.

## Conceitos abordados
- Declaração de vetor
- Laço de repetição `para`
- Entrada de dados (`leia`)
- Saída de dados (`escreva`, `escreval`)
- Soma acumulada de elementos

## Como usar
1. Abra o arquivo `Ex001.alg` no VisualG.
2. Execute o programa.
3. Digite 5 números conforme solicitado.
4. Observe os números digitados e a soma total exibida.

## Dicas
- Inicialize variáveis de soma antes do laço (`soma <- 0`) para evitar resultados incorretos.
- Preste atenção na indexação do vetor (posição 1 a 5).
