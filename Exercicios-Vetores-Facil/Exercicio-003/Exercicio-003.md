# Exercício 003 - Soma e média de elementos de um vetor

## Descrição
O programa lê **8 números inteiros** do usuário, calcula a **soma** e a **média** desses números, e exibe os resultados.  
Objetivo: treinar **vetores**, **laços de repetição** e **operações matemáticas básicas** em VisualG/Portugol.

## Conceitos abordados
- Declaração de vetor (`vetor[1..8] de inteiro`)  
- Laço de repetição `para`  
- Soma acumulada de elementos  
- Cálculo de média (`media <- soma / 8`)  
- Variáveis inteiras e reais (`inteiro`, `real`)  
- Saída formatada (`:4:2` para duas casas decimais)

## Como usar
1. Abra o arquivo `Ex003.alg` no VisualG.  
2. Execute o programa.  
3. Digite os 8 números solicitados.  
4. Observe a soma e a média exibidas na tela.
