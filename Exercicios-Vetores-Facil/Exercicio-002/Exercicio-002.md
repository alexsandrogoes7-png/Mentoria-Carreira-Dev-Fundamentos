# Exercício 002 - Soma de números pares em um vetor

## Descrição
O programa lê **10 números inteiros** do usuário, identifica os números pares e calcula a soma apenas desses números.  
Objetivo: treinar **vetores**, **laços de repetição** e **condicionais** em VisualG/Portugol.

## Conceitos abordados
- Declaração de vetor (`vetor[1..10] de inteiro`)  
- Laço de repetição `para`  
- Condicional `se ... então`  
- Operação de módulo `%` para identificar números pares  
- Soma acumulada de elementos  

## Como usar
1. Abra o arquivo `Ex002.alg` no VisualG.  
2. Execute o programa.  
3. Digite os 10 números solicitados.  
4. Observe a soma dos números pares exibida na tela.
