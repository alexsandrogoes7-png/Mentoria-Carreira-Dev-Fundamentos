# Exercícios de Lógica de Programação - Vetores

## Descrição
Este repositório contém exercícios de lógica de programação com **vetores**, desenvolvidos em **VisualG/Portugol**.  
O objetivo é treinar leitura, manipulação e operações com vetores de forma progressiva, do nível fácil ao difícil.

## Estrutura do Repositório

- `Exercicios-Vetores-Facil/`  
  Contém 5 exercícios iniciais, focados em conceitos básicos como leitura e exibição de vetores, soma e contagem de elementos.

- `Exercicios-Vetores-Medio/`  
  Contém 3 exercícios intermediários, envolvendo operações entre vetores, busca de elementos e ordenação simples.

- `Exercicios-Vetores-Dificil/`  
  Contém 2 exercícios avançados, com manipulação de vetores multidimensionais e lógica de indexação mais complexa.

Cada exercício possui:
- Código-fonte em VisualG (`.alg`)  
- README específico com explicação, conceitos abordados e instruções de uso  

## Como executar os exercícios

1. Abra o VisualG no seu computador.  
2. Navegue até a pasta do exercício desejado.  
3. Abra o arquivo `.alg`.  
4. Execute o programa e siga as instruções de entrada.  

## Observações

- Todos os exercícios são independentes.  
- É recomendado tentar resolver cada exercício sozinho antes de consultar o README específico ou a solução.
